title: "Software I use"
---

My [dotfiles](https://github.com/yagoyudi/dotfiles) contain various
configuration files. While I don’t use all of them regularly, I keep them
around in case I want to revisit or experiment with them. As a result, the
repository has become somewhat cluttered. So, to keep track of the tools I
actually use, I'm maintaining this list.

| Name | Purpose |  
|------|---------|  
| [gopass](https://www.gopass.pw/) | Secret management |  
| [chezmoi](https://www.chezmoi.io/) | Dotfiles management |  
| [wezterm](https://wezterm.org/index.html) | Terminal emulator and multiplexer |  
| [sway](https://swaywm.org/) | Window manager |  
| [qutebrowser](https://qutebrowser.org/) | Browser |  
| [zathura](https://pwmt.org/projects/zathura/) | Document viewer |  
| [NixOS](https://nixos.org/) | Linux distribution |  
| [nvim](https://neovim.io/) | Text editor |  
| [bash](https://en.wikipedia.org/wiki/Bash_(Unix_shell)) | Shell |  
| [cheat](https://github.com/yagoyudi/cheat) | Command reference tool |  
| [mako](https://github.com/emersion/mako) | Notification daemon |  
| [starship](https://starship.rs/) | Prompt |  
| [glow](https://github.com/charmbracelet/glow) | TUI markdown reader |  
| [pulsemixer](https://github.com/GeorgeFilipkin/pulsemixer) | Audio management |  
| [grim](https://github.com/emersion/grim) + [slurp](https://github.com/emersion/slurp) + [wl-clipboard](https://github.com/bugaevc/wl-clipboard) | Screenshot utilities |
